**SerialForge** is a modern serial communication utility designed for testing, debugging, and controlling FPGA-based systems through a clean and intuitive desktop interface. The software allows users to quickly detect available serial ports, connect to a selected device, send custom commands, and monitor real-time responses through a built-in console log.

Built with a developer-focused workflow in mind, SerialForge provides essential controls such as port refresh, connect and disconnect actions, command input, quick-send buttons, and timestamped serial output. Its modern dark-mode interface improves readability, reduces visual clutter, and makes hardware testing more efficient and professional.

SerialForge is ideal for FPGA developers, embedded systems engineers, students, and technicians who need a simple but reliable tool for serial communication, command testing, and device interaction.
